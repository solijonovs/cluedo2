# Cluedo Frontend

This folder contains a simple browser-based frontend for the Cluedo Part 2 project.

## Files
- `index.html` - Main frontend page
- `style.css` - Styling for the game interface
- `script.js` - Game logic for the browser version

## How to Run
Open `index.html` in a web browser.

No server or installation is required.

## Features
- Visual mansion board
- Room highlighting
- Movement buttons
- Suggestion system
- Refutation feedback
- Deduction notebook
- Accusation system
- New game button
